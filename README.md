# 📝 Todo App with Google Authentication

A modern **Todo Application** built with **Next.js 14**, **Firebase Authentication**, and **React Query**, featuring **Google Sign-In**, pagination, filtering, and local storage fallback.  

---

## ✨ Features

- 🔑 **User Authentication** – Sign in with Google using Firebase Auth.  
- ✅ **Manage Todos** – Add, edit, mark as complete, and delete tasks.  
- 🔍 **Search & Filter** – Quickly find todos and filter by status (All, Completed, Pending).  
- 📦 **Local Storage Fallback** – Works offline by storing todos locally if API is unavailable.  
- 📄 **Pagination** – Browse todos in pages for better usability.  
- 🎨 **Responsive UI** – Styled with **Tailwind CSS** + **Vanilla CSS** for clean design.  

---

## 🚀 Tech Stack

- [Next.js 14](https://nextjs.org/) – React Framework for the frontend  
- [React Query](https://tanstack.com/query) – Data fetching & caching  
- [Firebase](https://firebase.google.com/) – Google Authentication  
- [Tailwind CSS](https://tailwindcss.com/) + Vanilla CSS – UI Styling  
- [Axios](https://axios-http.com/) – HTTP client  
- [API](https://dummyjson.com/) – Mock API for todos (optional)
